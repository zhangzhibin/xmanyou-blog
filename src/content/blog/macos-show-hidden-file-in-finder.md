<!--kg-card-begin: markdown--><h2 id="">解决方法</h2>
<p>在Finder中，名字以.开头的文件或者文件夹默认是不显示的，要查看这些文件，除了在命令行查看外，还可以通过一个组合键来开启显示。</p>
<pre><code>shift + command + .
</code></pre>
