本文作者：优衣库的创始人，柳井正</p>
<p>经营者的四项能力：</p>
<h3 id="1">1. 创新的能力</h3>
<h4 id="11">1.1. 高远的目标</h4>
<p>如果目标过小，就很容易延续以往的惯性，也很容易被复制你的创意和方法，陷入艰难竞争。</p>
<p>不断提出更高的目标。</p>
<blockquote>
<p>从终点开始吧，因为只要你设定了终点，为了获得成功该做哪些事，就变得一目了然。<br>
-- 哈罗德.悉尼.吉宁 《职业经理人笔记》</p>
</blockquote>
<h4 id="12">1.2. 质疑常识，不受常识束缚</h4>
<p>人们容易把现有状态当做“常识”。很多对顾客无意义的事、给顾客带来不便的事，行业里的人，出于省事或者竞争空间的需要，却把它当做常识，或者视而不见。</p>
<p>努力突破这样的常识，为顾客带来便利，很可能发现潜在的商机。</p>
<h4 id="13">1.3. 树立高标准，不放松，不放弃</h4>
<p>持续满足顾客日益精进的要求。</p>
<p>先于顾客需求提高标准，顾客需要快马，主动提供汽车。</p>
<h4 id="14">1.4. 不畏风险，勇于尝试，敢于失败</h4>
<p><strong>追求稳定的公司，是不可能获得稳步发展的。</strong></p>
<h4 id="15">1.5. 严格要求，询问本质问题</h4>
<p>不要做善解人意的上司。</p>
<p>任由员工按照自己的标准做，员工就无法体验到真正意义上的成就感和自我成长。</p>
<p>鼓励员工，并承担最后的责任。</p>
<h4 id="16">1.6. 自问自答（自省）</h4>
<p>站在顾客的立场，用最挑剔的眼光来审视自己。</p>
<h4 id="17">1.7. 天外有天，不断学习</h4>
<p>学会与真正杰出的人交换信息。</p>
<h3 id="2">2. 赚钱的能力</h3>
<h4 id="21">2.1. 从心底希望顾客高兴</h4>
<ul>
<li>a). 让顾客感觉到惊喜</li>
<li>b). 了解顾客真正的心声</li>
<li>c). 提供资金真正认为优质的产品和店铺服务</li>
</ul>
<h4 id="22">2.2. 日复一日，完成必要的工作</h4>
<p>脚踏实地做好本职工作。</p>
<p>决定每一天的工作能否出色完成的，不是能力，而是习惯。</p>
<h4 id="23">2.3. 迅速实行</h4>
<p>迅速捕捉顾客需求，迅速商品化，并迅速提供给顾客。</p>
<p>只有时间，是人与人之间没有差别的。</p>
<p>迅速行动，赢得时间。</p>
<h4 id="24">2.4 现场，现实，现物</h4>
<p>不纸上谈兵。</p>
<p>解决问题最好的地方是现场。</p>
<h4 id="25">2.5. 集中解决问题</h4>
<p>要有舍弃的勇气，集中资源生产令我们自信的、最高标准的产品。</p>
<p>但是，如果从全局看，不做也不会有什么大事，那就不要浪费时间去做。</p>
<h4 id="26">2.6. 与矛盾做斗争</h4>
<p>成绩来自于解决矛盾的效果，矛盾越大，随之而来的利润就越高。</p>
<p>比如，提高质量，却同时降低成本。</p>
<h4 id="27">2.7. 做好准备，执着于成果而非计划</h4>
<p>优秀的人，实际做的事与计划不同，但是最后结果与目标一致。</p>
<p>平庸的人，实际做的事与计划一致，但是结果去没有达到。</p>
<h3 id="3">3. 建设团队的能力</h3>
<h4 id="31">3.1. 建立信赖关系</h4>
<p>构筑信赖的根基，是言行始终一致，信念始终如一。</p>
<h4 id="32">3.2. 全身心面对部下</h4>
<p>认真倾听，用心理解并接受部下之后，运用自己的经验、知识和能力进行分析，给他提出最好的意见和建议，而不是简单的认同他说的。</p>
<p>有时候做魔鬼，有时候做菩萨。</p>
<h4 id="33">3.3. 共享目标，责任到人</h4>
<p>不厌其烦地传递目标</p>
<p>一人一责</p>
<h4 id="34">3.4. 交付工作并予以评价</h4>
<p>不过度指挥，要有睁一眼闭一眼的勇气。</p>
<p>准确传达评价。</p>
<h4 id="35">3.5. 提出期望，发挥部下长处</h4>
<p>委以重任，提出期待。</p>
<p>最大限度发挥成员的优势，不要盯着缺点。</p>
<h4 id="36">3.6. 积极肯定多样性</h4>
<p>放弃偏执，克服先入为主，适时入乡随俗。</p>
<h4 id="37">3.7. 抱持最强烈的取胜欲望，坚持自我变革</h4>
<p>身先士卒，不断自我完善，坚持自我管理。</p>
<h3 id="4">4. 追求理想的能力</h3>
<h4 id="41">4.1. 要有使命感</h4>
<p>一味追求金钱，金钱反倒会离你而去。</p>
<p>只有为社会做贡献，才能得到社会的认可，才会被允许存在。</p>
<h4 id="42">4.2. 使命感赋予我们的东西</h4>
<p>责任感，职业道德，主观能动性，吸引力</p>
<h4 id="43">4.3. 与使命感的绊脚石作斗争</h4>
<p>使命感的绊脚石：</p>
<ul>
<li>a). 自我为中心，高高在上的态度</li>
<li>b). 使工作流于没有特色、模仿性的工作</li>
<li>c). 机械、缺乏独创性、生搬硬套的思考模式</li>
<li>d). 官僚主义</li>
<li>e). 评价标准宽松，以实力之外的因素来决定人事安排</li>
</ul>
